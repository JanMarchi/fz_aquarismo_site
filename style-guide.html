♿ Acessibilidade

Conformidade WCAG 2.1 AA

  • Contraste: Mínimo 4.5:1 para texto normal
  • Teclado: Todos os elementos navegáveis via Tab
  • ARIA: Labels semânticos e roles
  • Screen Reader: Testado com NVDA e VoiceOver
  • Sem dependência de cor: Informação transmitida por cor + ícone/texto
  • Redução de movimento: Respeita prefers-reduced-motion
